# OpenAI API

**Project:** 					      Use the OpenAI API to apply AI capabilities to Excel sheets

**Consumer:**					      CCC EHSD

**Program languages:**  VBA

**Description:**        The idea is to use complex data transformations in Excel based on a specified template, which can be easily created using AI. The attached video demonstrates some of the possible data processing techniques on Excel sheets. The repository also contains my source code:

 - CallOpenAI.vba - VBA function code for OpenAI API usage
 - OpenAI_integrations_with_Excel.mp4 - video demonstration of how to use the function
 
**Results / Key Findings:** The implementation of the CallOpenAI function in Excel workbooks has made it easy to leverage the AI capabilities for preparing and analyzing tabular data.
